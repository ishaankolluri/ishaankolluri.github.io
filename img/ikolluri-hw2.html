df = pd.read_excel(
    "Bank failure data.xlsx", 
    names=["name", "qtr", "tierone", "texas", "size", "deposits", "nco", "loans", "change_mix", "cre_assets", "lb_assets", "securities", "failed", "cert"],
)

def labeling(val):
    return 1 if val == "Yes" else 0

def quarter_time_series(quarter_str):
    quarters = quarter_str.split("Q")
    year = float(quarters[0])
    quarter = (float(quarters[1]) - 1) / float(4)
    return year + quarter

df['qtr_time'] = df['qtr'].apply(quarter_time_series)
df['labels'] = df['failed'].apply(labeling)
df = df.drop('failed', 1)
df = df.drop('qtr', 1)

Question: What are the top two predictors of bank failure?¶

Based on the ANOVA p-value calculation for each feature (excluding cert, name, and time/quarter), where the smaller p values indicate more evidence in support of failure, the construction and land development loan exposure + the nonperforming commercial real estate to assets features seem to be the most discriminatory features.

I also computed the correlation to bank failure for each one. Similarly, loans & real estate to assets have the highest correlation coefficients, numbering at 0.4367... and 0.4039..., respectively.

Note this can be confounded by how the data is handled for NaNs. I chose to insert the average of the bank's quarterly values.
